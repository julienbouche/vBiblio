amazon_browsenodes

Returns an array of a modes' valid browsenodes

amazon_browsenodes(str mode)

This function returns an associative array with the valid amazon™ browsenodes of a mode as keys and their common names as value (f.e. "1000" as key and "Book Bestsellers" as value). You can get the valid amazon™ modes with the function amazon_modes().

Example 1. Get browsenodes of the mode "books", sort and display them:

$Browsenodes=amazon_browsenodes("books");
asort($Browsenodes);
while(list($Key,$Value)=each($Browsenodes)) {
  echo "<a href='displaycategory.php?browsenode=$Key'>$Value</a>\n";
 }
