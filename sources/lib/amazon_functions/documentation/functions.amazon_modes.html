amazon_modes

Returns an array of the valid amazon modes

amazon_modes()

This function needs no vars and simply returns an associative array with the valid amazon™ mode names as key and their common names as value (f.e. "universal" as key and "Tools & Hardware" as value). You will need these modes f.e. for the function amazon_search_manufacturer().

Example 1. Get modes, sort and display them:

$Modes=amazon_modes();
asort($Modes);
while(list($Key,$Value)=each($Modes)) {
  echo "<a href='displaymode.php?mode=$Key'>$Value</a>\n";
 }
