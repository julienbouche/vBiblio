Filzhut.net amazon_functions for PHP (Version 0.9)
Making the work with Amazon.com™ Web Services Version 1.0 much easier

Include

To get access to the amazon_functions you just have to include the file "amazon_functions.php" to the PHP-Script in which you want to use the functions. Including this file is very easy as the following example might illustrate.

Example 1. Include the functions to your script:

<?php

include('functions/amazon_functions.php');

... (start programming)

php?>
